# Report: "FCAJ Meetup 06/06/2026"

### Event Objectives

The purpose of this meetup was to bring the tech community together and create a space where participants could learn from speakers with real-world system and cloud experience. Through topics such as networking, Docker, web security, Machine Learning, and career development in system-related roles, the event helped attendees refresh practical knowledge, learn from real cases, and connect with others who share the same direction.

### Speakers

* **Nguyễn Quốc Bảo** - Swinburne University of Technology
* **Bảo Huỳnh** - Junior Cloud Native Developer, Endava Vietnam
* **Lê Hoàng Gia Đại** - Ho Chi Minh City University of Technology
* **Vinh Trần** - System Administrator, Central Retail Group

### Highlights

#### Connecting Godot clients with AWS WebSockets

This session helped me review basic networking knowledge and understand how clients communicate with servers in a real-time application. The speaker also introduced ENet in Godot and demonstrated a rock-paper-scissors game that used several AWS services.

What I found most useful was the explanation of player matching, connection handling, and disconnect recovery. It gave me a clearer view of how WebSocket supports two-way communication in applications that need fast responses.

#### Docker and virtualization

The speaker explained how Docker is used to package and deploy applications. The talk started with the value of virtualization, then moved into Docker basics, common commands, and how to organize a runtime environment for applications.

One especially helpful topic was caching in Docker, including Docker layer caching, which can reduce image build time. The simple demos made it easier for me to understand how Docker fits into day-to-day development.

#### WAF and Machine Learning for cyber attack detection

This section focused on combining AWS WAF with Machine Learning to improve cyber attack detection. Instead of relying only on static rules, the solution can help a system defend itself more proactively and adapt better to new threats.

The demo made the workflow easier to picture, especially how a security system can collect signals, analyze behavior, and respond in a way that reduces risk for web applications.

#### Career journey from IT Helpdesk to Senior Sysadmin

The talk by Vinh was memorable because it felt very close to real working life. He shared that he did not come from a top university, but through persistence, hands-on practice, and steady experience from many projects, he grew from an IT Helpdesk role to Senior Sysadmin.

He also talked about interviews at large companies, production incidents such as a hard drive failure causing downtime, and how he stayed calm while analyzing root causes step by step. The session also gave useful guidance on a Cloud learning path for beginners.

### What I Learned

* Refreshed my knowledge of networking, WebSocket communication, and how clients talk to servers in real-time apps.
* Better understood Docker’s role in development and deployment, especially Docker layer caching for faster image builds.
* Learned how AWS WAF can work with Machine Learning to improve attack detection and mitigation.
* Saw how practical experience, troubleshooting skills, and calm decision-making matter a lot in system operations work.
* Understood that building real projects is often more effective than learning theory alone.

### How I Can Apply It

* Use Docker to build more consistent development environments across machines.
* Optimize Dockerfiles and take advantage of layer caching to shorten image build times in personal projects and assignments.
* Use Docker Compose to spin up the required services quickly in development.
* When troubleshooting system issues, break the problem into steps, verify assumptions, and stay calm before jumping to conclusions.
* Practice more with WebSocket-based flows and AWS services to better understand how real systems behave.

### Event Experience

#### Learning from the speakers

All speakers used practical examples and real stories instead of only theory. That made it easier for me to understand how Docker, WebSocket, WAF, and Machine Learning are used in a business environment.

The career story shared by Vinh was especially inspiring because it showed that your starting point is not the only thing that matters. Persistence, curiosity, and continuous learning are just as important.

#### Lessons learned

Foundational knowledge is necessary, but hands-on practice and real incident handling are what truly strengthen skills. I also realized that when dealing with systems, it is better to analyze issues step by step than to rush toward a quick answer.

Meetups like this are also a great chance to expand knowledge, follow technology trends, and learn from people already working in the field.
