* Learn and practice Serverless architecture with Lambda, API Gateway, SQS, SNS.

* Mastered usage and connection of 5 new AWS services (Lambda, API Gateway, SQS, SNS, CloudFront).

* Better understood Serverless model and low operational overhead architecture design.
