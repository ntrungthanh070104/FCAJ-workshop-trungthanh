#### Goal

Enable AI/voice services so the system can analyze CVs, generate/score interview questions, and support microphone input.


#### Services used

| Service | Role |
| --- | --- |
| Amazon Bedrock | Analyze CVs, generate questions, score answers. |
| Amazon Polly | Create interview question audio. |
| Amazon Transcribe | Convert answer audio to text. |
| Amazon S3 | Store audio and transcripts. |
