#### Mục tiêu

Kiểm thử end-to-end để đảm bảo frontend, backend và AWS services hoạt động đồng bộ.


#### Nhóm kiểm thử

| Nhóm | Nội dung |
| --- | --- |
| User flow | Login, Upload CV, Analyze CV, AI Interview, Result, History. |
| Voice flow | Polly audio, microphone recording, Transcribe transcript. |
| Admin flow | Users, CVs, Interviews, Review Queue, Audit Log, Export CSV. |
| Operation | CloudWatch Logs, CORS, IAM, Budget. |
